# HackVibe - 黑客松现场社交产品原型

> 将"填表社交"转化为"现场 RPG 游戏"的赛博朋克风格原型

## 🎮 项目概述

HackVibe 是一个专为黑客松现场设计的社交产品原型，采用赛博朋克美学风格，通过游戏化的交互方式降低社交压力，帮助参赛者快速发现有趣的人和匹配技能。

## 📁 文件结构

```
gemini-ask/
├── index.html          # 首页/着陆页
├── onboarding.html     # 灵魂生成器（注册页）
├── galaxy.html         # 现场星图（主页）
├── profile.html        # 黑客身份卡（详情页）
├── gacha.html          # 命运扭蛋机（匹配页）
├── styles.css          # 全局样式（赛博朋克风格）
└── README.md           # 本文件
```

## 🚀 快速开始

### 本地运行

1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器：
   ```bash
   # 使用 Python 3
   python -m http.server 8000

   # 使用 Node.js
   npx serve
   ```
3. 在浏览器中访问 `http://localhost:8000`

### 体验流程

1. **首页** (`index.html`) - 了解产品概念
2. **灵魂生成器** (`onboarding.html`) - 创建你的赛博人设
3. **现场星图** (`galaxy.html`) - 浏览在线用户
4. **黑客身份卡** (`profile.html`) - 查看用户详情
5. **命运扭蛋机** (`gacha.html`) - 获取 AI 推荐

## 🎨 设计特色

### 视觉风格

- **赛博朋克美学**：霓虹色彩、扫描线、故障效果
- **独特字体选择**：
  - `Press Start 2P` - 像素风格标题
  - `Orbitron` - 科技感正文
  - `VT323` - 复古终端风格
- **动态效果**：
  - 霓虹光晕动画
  - 扫描线滚动
  - 全息卡片效果
  - 浮动/脉冲动画

### 核心功能

#### 1. 灵魂生成器 (Onboarding)
- 极简三步注册：昵称 + 技术栈 + 心情
- AI 生成 3 个中二称号供选择
- 生成赛博风格"黑客身份卡"

#### 2. 现场星图 (Galaxy)
- 气泡图可视化用户分布
- 雷达扫描动画效果
- 实时技能筛选
- 状态广播功能
- 气泡大小表示匹配度

#### 3. 黑客身份卡 (Profile)
- 全息风格个人名片
- 称号 + 技能 + 需求展示
- AI 破冰话题推荐
- 贴纸低压力社交
- 弹幕装饰效果

#### 4. 命运扭蛋机 (Gacha)
- 3D 扭蛋机视觉效果
- 摇一摇/拉杆交互
- SSR/SR/R 稀有度系统
- 匹配理由展示
- 算命式趣味文案

## 🎯 设计原则

### 核心理念
1. **去严肃化** - 拒绝 LinkedIn 式的职业假笑
2. **即时满足** - 所见即所得，推荐即连接
3. **极简交互** - 赶代码的人没时间填长表单
4. **趣味优先** - 用"梗"和"标签"代替简历

### 实现细节

- **色彩系统**：
  - 霓虹青 `#00ffff` - 主色调
  - 霓虹粉 `#ff00ff` - 强调色
  - 霓虹紫 `#bf00ff` - 辅助色
  - 霓虹绿 `#00ff41` - 在线状态
  - 深色背景 `#0a0e27` - 基础背景

- **动画策略**：
  - CSS-only 动画优先
  - 使用 `animation-delay` 实现错开效果
  - 关键时刻的大动画（页面加载、抽卡）
  - 微交互：hover、点击反馈

- **响应式设计**：
  - 移动端优先
  - 断点：768px
  - 触摸友好的按钮尺寸

## 💡 技术栈

- **纯前端实现**：HTML5 + CSS3 + Vanilla JavaScript
- **无依赖**：不需要任何框架或库
- **本地存储**：使用 `localStorage` 保存用户数据
- **Google Fonts**：
  - Orbitron
  - VT323
  - Press Start 2P

## 🚧 原型限制

这是一个**高保真交互原型**，不包含：

- ❌ 后端服务器
- ❌ 真实数据库
- ❌ 用户认证系统
- ❌ 实时通讯（IM）
- ❌ 真实的 AI 称号生成

但包含：

- ✅ 完整的 UI/UX 流程
- ✅ 交互动画和效果
- ✅ 响应式布局
- ✅ 本地数据持久化
- ✅ 模拟的用户匹配逻辑

## 🎨 自定义配置

### 修改颜色主题

编辑 `styles.css` 中的 CSS 变量：

```css
:root {
  --neon-cyan: #00ffff;      /* 主色调 */
  --neon-pink: #ff00ff;      /* 强调色 */
  --neon-purple: #bf00ff;    /* 辅助色 */
  --dark-bg: #0a0e27;        /* 背景色 */
  /* ... */
}
```

### 添加模拟用户

编辑各页面的 JavaScript 中的用户数组：

```javascript
const mockUsers = [
  { name: '你的昵称', skills: ['技能1', '技能2'], ... },
  // 添加更多用户
];
```
