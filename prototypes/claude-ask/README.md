# 人间观察 - 黑客松社交实验原型

> 一个让黑客松参与者在 3 分钟内被看见、被理解、被连接的趣味社交工具

## 快速开始

直接在浏览器中打开 `index.html` 即可体验完整原型。

## 产品功能

### 1. 灵魂拷问
- 3个精心设计的问题
- 收集用户的人格特质、社交偏好和技能信息
- 趣味化的loading体验

### 2. 我的名片
- AI生成的个性化画像
- 包含标题、三句话简介、技能标签
- 神秘指数评分系统
- 可重新生成描述

### 3. 人间观察
- 瀑布流名片墙
- 支持按技能标签筛选
- 点击卡片查看详情
- 6个示例人物卡片

### 4. 今日推荐
- 3种推荐逻辑：技能互补、气质相似、随机惊喜
- 每个推荐附带推荐理由
- 可查看推荐对象的完整名片

### 5. 命运转盘
- 随机匹配功能
- 带有旋转动画的转盘交互
- 神秘感十足的彩蛋页面

## 设计特色

### 视觉风格
- **深夜观察室美学**：神秘、温暖、人性化
- **独特字体组合**：Piazzolla（优雅衬线）+ JetBrains Mono（代码感）
- **色彩方案**：深色基底 + 紫色荧光（#7C3AED）+ 绿色点缀（#10B981）
- **视觉细节**：噪点纹理、渐变背景、玻璃态模糊效果

### 交互体验
- 流畅的页面切换动画（300ms）
- 趣味化的文案（"正在解析你的灵魂碎片..."）
- 微交互反馈（按钮hover、缩放、颜色变化）
- 模态框展示详细信息

### 设计原则
- **有趣 > 有用**：用游戏化设计降低社交焦虑
- **轻量 > 完整**：5个核心页面解决所有功能
- **人味 > 机味**：AI在后台，用户看到的是有趣的成品

## 技术实现

- 纯HTML/CSS/JavaScript实现
- 无需构建工具，直接运行
- 响应式设计，支持移动端
- 使用CSS变量管理主题
- 动画使用CSS transition和animation
